# Camera Color Calibration — 产品特性点

**功能模块**：Professional Calibration Tool > Camera Color Calibration
**平台**：TCL OS · 中国区 · 1920×1080
**版本**：Demo v1.0 · 2026-07-20

---

## 一、入口与导航

### 1.1 工具入口页（Launcher）
- 苹果风格黑色全屏入口，顶部导航栏含 TCL 红色 Logo + "Picture Calibration" 标题
- 工具卡片展示：蓝色渐变相机图标、功能名称、功能描述、绿色「可用」状态指示
- 遥控器方向键导航 + Enter 进入工具，底部操作提示常驻显示

### 1.2 全局导航规则
- 完整遥控器 D-pad 支持：↑ ↓ ← → 方向键空间导航
- Enter 键确认，Escape 键逐级返回（工具页 → 入口页）
- 弹窗弹出时自动锁定焦点（Focus Trap），关闭后焦点还原
- 页面切换带淡入动效，防止视觉跳变

---

## 二、校准模式

### 2.1 模式切换 Tab（标准 / 专业）
- 首页顶部药丸形 Tab 切换，标准对应快速校准，专业对应导入校准
- 两种校准模式**互斥**，同时只能有一种生效
- 已有生效校准时切换 Tab，弹出互斥确认弹窗，需用户明确确认后才切换
- 面板高度固定，切换 Tab 时不发生尺寸跳变

---

## 三、标准模式（快速校准）

### 3.1 校准强度 Slider
- 校准强度范围 Level 1–5，默认值 Level 3
- 遥控器左右键实时调节数值，聚焦时蓝色边框高亮发光
- 滑条填充与滑块位置实时联动，右侧数字实时更新

### 3.2 生效范围切换
- 左右箭头切换组件：**长期生效 / 仅本次开机 / 关闭** 三态循环
- 遥控器左右键或点击箭头均可切换，切换后 Toast 轻提示当前状态

### 3.3 应用 / 取消
- 点击「应用」后按钮变为红色「取消」，Toast 提示校准已应用
- 再次点击「取消」恢复蓝色「应用」，清除校准状态
- 应用校准仅在当前界面给出提示，不跳转页面

---

## 四、专业模式（导入校准）

### 4.1 导入前状态
- 展示功能说明与「开始导入校准」按钮
- 生效范围切换行（与标准模式共享同一变量）
- 「删除并重置」快捷按钮（需二次确认）

### 4.2 拍摄测试图页
- **全屏展示**四色块测试图（红 / 绿 / 蓝 / 灰），四角带像素风定位标记
- 右下角半透明胶囊提示「拍摄完成后，请按【OK】键进入下一步」
- 按 Enter/OK 键直接进入下一步，无需点击按钮

### 4.3 扫码上传页
- 展示模拟二维码 + 扫描动画光条
- 初始提示引导用户扫码并按步骤上传
- 点击「模拟扫码」：
  - 新标签页打开真实校准官网（`https://yangshuai990529.github.io/color-correction/`）
  - 提示文字变为蓝色高亮引导说明
  - 「一键拉取云端文件」按钮激活

### 4.4 云端拉取页
- 圆形 Loading 动画 + 进度条（3 秒模拟）
- 状态文字分阶段显示：「连接云端服务器…」→「已检测到云端文件，正在分析拉取中」
- 自动完成后跳转校准管理页

### 4.5 校准管理页
- 展示校准文件信息：文件名称、生成时间、适用设备、当前状态、校准方式
- 生效范围切换（同标准模式，三态循环）
- 「应用并使用当前校准文件」主按钮 → 返回首页并 Toast 提示
- 「删除并重置」/ 「重置校准设置」按钮，均需二次确认弹窗

### 4.6 导入成功后状态
- 专业 Tab 自动切换为「已导入校准文件」状态
- 展示：校准状态（✓ 校准已生效）、生效时间（实时写入）、生效文件名
- 「应用」按钮初始显示为红色「取消」（已生效状态）
- 「开始导入校准」按钮变为「重新导入校准」
- 执行删除/重置后，所有状态恢复初始

| 取消导入校准 | 按钮直接切换，Toast 提示 |

## 六、视觉与交互规范

### 6.1 背景与面板
- 首页：全屏 SVG 雪山风景背景（无蒙层），左上角半透明毛玻璃面板（760×860px）
- 其他页面：深色渐变背景，内容居中毛玻璃卡片
- 面板背景：`rgba(14,17,35,0.82)` + `backdrop-filter: blur(24px)`

### 6.2 焦点状态
- 聚焦态：蓝色边框（`#4b7bff`）+ 蓝色发光阴影 + 轻微放大（scale 1.04）
- Slider 聚焦：蓝色边框 + 蓝色背景光晕
- 生效范围切换器聚焦：蓝色边框高亮

### 6.3 Toast 轻提示
- 所有关键操作均有 Toast 反馈，2.2 秒后自动消失
- 弹窗与 Toast 均在 `#stage` 内部，随画布等比缩放，无偏移

### 6.4 分辨率适配
- 画布固定 1920×1080，JS 动态计算缩放比例适配任意窗口尺寸
- `scale = Math.min(window.innerWidth / 1920, window.innerHeight / 1080)`

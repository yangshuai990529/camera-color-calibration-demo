# Professional Calibration Tool — Camera Color Calibration

TCL OS 中国区电视端相机色彩校准交互原型，共 7 个页面，支持快速校准与导入校准两条完整操作路径。

## 运行方式

1. 打开本目录。
2. 双击 `index.html`，使用浏览器直接运行。
3. 无需安装依赖、启动服务或连接真实电视设备。
4. 建议使用最新版 Chrome、Edge 或 Safari，并切换至全屏模式体验电视端效果。

## 支持的遥控器按键

原型使用电脑键盘模拟电视遥控器：

| 键盘按键 | 遥控器含义 | 用途 |
|---|---|---|
| `ArrowUp` / `ArrowDown` | 上 / 下 | 在纵向控件间移动焦点 |
| `ArrowLeft` / `ArrowRight` | 左 / 右 | 在横向控件、档位或操作按钮间移动焦点 |
| `Enter` | OK / 确定 | 激活当前焦点项、提交操作或进入下一页 |
| `Escape` | Back / 返回 | 关闭弹窗、取消当前操作或返回上一页 |

## 页面与交互说明

### 1. 首页

- 提供“快速校准”和“导入校准”两个入口。
- 使用方向键切换焦点，按 `Enter` 进入对应流程。
- 两种校准方式互斥；完成其中一种后，另一种方式不会同时处于生效状态。

### 2. 快速校准页

- 提供 `Level 1` 至 `Level 5` 五个校准档位。
- 使用方向键选择档位，当前档位显示选中状态。
- 聚焦“应用”并按 `Enter` 后进入拍摄引导页。
- 未切换档位时沿用页面默认档位。

### 3. 拍摄引导页

- 展示测试图拍摄说明和操作提示。
- 按说明准备相机并拍摄电视画面。
- 聚焦“OK”按钮并按 `Enter`，进入二维码页。
- 按 `Escape` 返回快速校准页，并保留已选档位。

### 4. 二维码页

- 使用手机扫描二维码，按移动端指引上传拍摄结果。
- 上传完成后，聚焦“已完成上传”并按 `Enter`。
- 原型不连接真实上传服务，按钮用于模拟上传完成状态。

### 5. 云端分析页

- 自动播放约 3 秒的云端分析进度动画。
- 分析期间无需操作，完成后自动进入结果页面。
- 快速校准流程完成后生成并应用对应校准结果。

### 6. 导入成功页

- 从首页选择“导入校准”后，模拟从云端拉取校准文件。
- 拉取成功后展示文件摘要与“校准详情”入口。
- 聚焦详情入口并按 `Enter`，进入校准管理页。

### 7. 校准管理页

- 查看校准文件名、导入时间、校准方式及生效范围。
- “生效范围”使用 Toggle 控件切换启用状态或适用范围状态。
- “删除”用于移除当前校准文件；执行前需确认。
- “重置”用于恢复默认校准配置；执行前需确认。
- 弹窗打开后，焦点限制在弹窗按钮内；确认或取消后恢复至原触发项。

## 注意事项

- 本项目为交互演示原型，不连接真实相机、手机、二维码上传或云端分析服务。
- 二维码、文件拉取、分析进度及校准结果均为模拟数据，不代表真实测量结果。
- 建议通过键盘模拟遥控器操作；鼠标操作仅作为调试补充。
- 页面任一时刻应保持唯一清晰焦点；若浏览器焦点不在页面内，请先单击页面空白处再使用方向键。
- `Escape` 优先关闭当前弹窗；无弹窗时返回上一页。首页按 `Escape` 不执行破坏性操作。
- 云端分析动画播放期间请等待自动完成，避免重复触发操作。
- 删除和重置属于结果变更操作，必须在确认弹窗中二次确认。
- 原型数据仅保存在当前浏览器会话中；刷新或关闭页面后可能恢复默认状态。

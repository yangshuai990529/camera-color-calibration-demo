# Professional Calibration Tool — Camera Color Calibration 测试用例

## 1. 测试范围

- TCL OS 中国区电视端 7 页交互原型。
- 快速校准、导入校准、互斥切换、校准管理和异常返回流程。
- 键盘模拟遥控器：方向键、`Enter`、`Escape`。

## 2. 测试前置条件

- 双击 `index.html`，浏览器可正常打开原型。
- 浏览器窗口具有键盘焦点。
- 除特殊用例外，初始状态为“未校准”。
- 原型中的上传、云端分析及文件拉取均按模拟行为验证。

## 3. 正常流测试用例

### 3.1 快速校准完整流

| 用例 ID | 测试目的 | 前置条件 | 操作步骤 | 预期结果 |

| QC-004 | 确认拍摄引导 | 位于拍摄引导页 | 1. 检查测试图及说明 2. 聚焦“OK” 3. 按 `Enter` | 测试图和说明完整显示；进入二维码页 |

## 8. 通用验收标准

- 任一页面始终只有一个清晰可见的焦点态。
- 所有可操作控件均可仅通过方向键与 `Enter` 到达并激活。
- `Escape` 遵循“关闭弹窗优先、返回上一页其次”的规则。
- 快速校准与导入校准绝不同时生效。
- 未完成、取消或失败的流程不得覆盖已有有效校准数据。
- 删除和重置必须经过二次确认，取消后不得产生数据变更。
- 进度动画、页面跳转和确认操作不得因重复按键被重复执行。
- 全流程无浏览器脚本错误、焦点丢失、不可恢复页面或无响应状态。

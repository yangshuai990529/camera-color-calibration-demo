# Professional Calibration Tool — Camera Color Calibration 产品规格

## 1. 产品背景

专业画质调校人员需要通过相机拍摄电视测试图并结合云端分析结果，完成电视色彩校准。现有流程涉及档位选择、拍摄、手机上传、云端分析及校准文件管理，需要在 TCL OS 中国区电视端形成清晰、可回退、可管理的闭环体验。

本原型覆盖 7 个页面，支持“快速校准”和“导入校准”两种互斥方式，并提供校准结果的生效范围管理、删除和重置能力。

## 2. 产品目标

- 通过电视遥控器完成端到端色彩校准操作。
- 降低测试图拍摄和手机上传流程的理解成本。
- 明确展示云端分析进度与校准结果。
- 支持导入已有校准文件，减少重复拍摄。
- 确保快速校准与导入校准不会同时生效，避免配置冲突。
- 提供删除、重置和生效范围控制，形成可维护的校准资产管理闭环。

## 3. 用户与使用环境

- **目标用户**：专业调校人员、售后工程师、具备专业校准需求的高级用户。
- **区域与系统**：中国区，TCL OS 电视端。
- **输入设备**：电视遥控器方向键、OK 键和返回键。
- **原型约束**：离线演示，不接入真实相机、上传、云端分析和文件服务。

## 4. 核心流程

### 4.1 快速校准

首页 → 快速校准页 → 拍摄引导页 → 二维码页 → 云端分析页 → 校准管理页。

### 4.2 导入校准

首页 → 导入成功页（拉取文件）→ 校准详情 → 校准管理页。

## 5. 功能规格

### 5.1 首页

#### 字段与控件

| 字段/控件 | 类型 | 说明 |
|---|---|---|
| 页面标题 | 文本 | Camera Color Calibration |
| 快速校准 | 入口卡片 | 进入档位选择及拍摄分析流程 |
| 导入校准 | 入口卡片 | 拉取并导入已有校准文件 |
| 当前校准摘要 | 状态文本 | 可展示当前方式、生效状态或“未校准” |

#### 页面状态

- 未校准：两个入口均可进入。
- 快速校准已生效：显示快速校准摘要；导入后将替换当前结果。
- 导入校准已生效：显示导入文件摘要；重新快速校准后将替换当前结果。
- 入口聚焦：卡片呈现高亮焦点态。

#### 业务规则

- 两个入口只代表可选择的流程，不表示两种方式可同时生效。
- 按 `Enter` 进入聚焦入口；按 `Escape` 在首页不触发删除、退出或状态清空。
- 若切换校准方式会覆盖当前生效配置，应在提交新结果前明确提示或执行替换规则。

### 5.2 快速校准页

#### 字段与控件

| 字段/控件 | 类型 | 说明 |
|---|---|---|
| Level 1–5 | 单选档位 | 五个互斥校准强度档位 |
| 当前档位 | 状态 | 高亮显示唯一选中值 |
| 应用 | 按钮 | 确认档位并进入拍摄引导 |

#### 页面状态

- 默认状态：预选系统默认档位。
- 档位聚焦：焦点与已选状态需视觉区分。
- 档位已选：任一时刻仅一个 Level 处于选中状态。
- 应用提交：记录所选档位并进入下一页。

#### 业务规则

- `Level 1` 至 `Level 5` 为互斥单选。
- 切换焦点不等于修改档位；按 `Enter` 后才更新选中档位。
- “应用”使用当前选中档位生成本次快速校准任务。
- 从拍摄引导页返回时保留本次已选档位。

### 5.3 拍摄引导页

#### 字段与控件

| 字段/控件 | 类型 | 说明 |
|---|---|---|
| 测试图 | 图像/示意 | 展示需拍摄的测试画面 |
| 拍摄说明 | 文本 | 描述相机位置、取景及拍摄要求 |
| OK | 按钮 | 确认已完成准备并进入二维码页 |

#### 页面状态

- 默认状态：展示完整测试图和拍摄说明，焦点落在“OK”。
- 返回状态：返回快速校准页并恢复原焦点与已选档位。

#### 业务规则

- 页面不校验真实拍摄动作。
- 只有激活“OK”后才进入二维码页。
- 说明文字不得遮挡测试图关键区域。

### 5.4 二维码页

#### 字段与控件

| 字段/控件 | 类型 | 说明 |
|---|---|---|
| 二维码 | 图像 | 用于手机扫码进入上传流程 |
| 上传说明 | 文本 | 提示扫码、选择照片及上传 |
| 已完成上传 | 按钮 | 用户确认移动端上传完成 |

#### 页面状态

- 等待扫码/上传：默认状态。
- 上传确认：激活“已完成上传”后创建分析任务。
- 上传失败：提示检查网络或重新上传，并允许重试。

#### 业务规则

- 原型中的二维码和上传状态均为模拟。
- 未激活“已完成上传”不得进入云端分析页。
- 重复按 `Enter` 只创建一个分析任务，防止重复提交。

### 5.5 云端分析页

#### 字段与控件

| 字段/控件 | 类型 | 说明 |
|---|---|---|
| 分析状态 | 文本 | 如“正在分析色彩数据” |
| 进度 | 进度动画 | 约 3 秒完成 |
| 结果状态 | 状态 | 成功、失败或重试中 |

#### 页面状态

- 分析中：进度从 0% 递增至 100%。
- 分析成功：生成快速校准结果，替换原生效方式并自动进入管理页。
- 分析失败：显示失败原因摘要及“重试/返回”操作。
- 重试中：重新开始进度，禁止并发提交。

#### 业务规则

- 动画时长约 3 秒，仅用于原型演示。
- 分析中应阻止重复启动任务。
- 仅成功结果可写入当前校准数据。
- 失败不得覆盖原有有效校准文件。

### 5.6 导入成功页

#### 字段与控件

| 字段/控件 | 类型 | 说明 |
|---|---|---|
| 拉取状态 | 状态 | 拉取中、成功、失败 |
| 校准文件名 | 文本 | 拉取到的文件名称 |
| 导入时间 | 文本 | 文件成功导入的本地时间 |
| 校准详情 | 入口按钮 | 查看并管理导入结果 |
| 重试 | 按钮 | 拉取失败时重新请求 |

#### 页面状态

- 拉取中：展示加载状态，不允许重复拉取。
- 拉取成功：展示文件摘要和“校准详情”。
- 拉取失败：保留原校准状态，提供重试或返回。

#### 业务规则

- 拉取成功后，将校准方式标记为“导入校准”。
- 新导入文件替换当前快速校准结果或旧导入文件。
- 文件详情未成功解析时不得标记导入成功。
- 按 `Escape` 返回首页；失败或取消时不得改变原生效配置。

### 5.7 校准管理页

#### 字段与控件

| 字段/控件 | 类型 | 说明 |
|---|---|---|
| 校准文件名 | 文本 | 当前校准文件唯一可读名称 |
| 导入时间 | 日期时间 | 当前文件创建或导入时间 |
| 校准方式 | 枚举文本 | 快速校准 / 导入校准 |
| 生效范围 | Toggle | 控制当前校准配置的启用/适用状态 |
| 删除 | 危险操作按钮 | 删除当前校准文件 |
| 重置 | 次级危险操作按钮 | 恢复默认校准配置 |
| 确认弹窗 | Dialog | 对删除、重置进行二次确认 |

#### 页面状态

- 已生效：Toggle 开启，当前文件参与画质校准。
- 未生效：Toggle 关闭，保留文件但不应用。
- 删除确认：焦点锁定在“取消/删除”。
- 重置确认：焦点锁定在“取消/重置”。
- 删除成功：清空当前校准数据，返回未校准状态。
- 重置成功：恢复系统默认校准值，并清理自定义生效状态。

#### 业务规则

- Toggle 切换立即更新生效状态，但不删除校准文件。
- 删除必须二次确认；取消后数据不变，焦点返回“删除”。
- 重置必须二次确认；取消后数据不变，焦点返回“重置”。
- 删除后文件详情不可继续访问。
- 重置后首页显示默认或未校准状态，具体以原型展示为准。

## 6. 互斥逻辑

### 6.1 校准方式互斥

- 当前有效校准方式只能是以下三种状态之一：`NONE`、`QUICK`、`IMPORTED`。
- 快速校准分析成功后，将状态设为 `QUICK`，并替换当前导入校准结果。
- 导入文件成功后，将状态设为 `IMPORTED`，并替换当前快速校准结果。
- 流程进入、取消或失败不改变原有有效方式；只有成功提交结果后才切换。

### 6.2 档位互斥

- `Level 1–5` 任一时刻只能选择一个。
- 新档位选中后，原档位立即取消选中。

### 6.3 操作互斥

- 云端分析期间不可重复提交上传完成。
- 文件拉取期间不可重复发起导入。
- 删除或重置弹窗显示期间，页面其他控件不可获得焦点或执行操作。

## 7. 异常场景处理

| 异常场景 | 处理规则 |
|---|---|
| 上传未完成即尝试继续 | 停留二维码页，提示先完成上传 |
| 上传失败 | 展示失败提示，允许重试或返回 |
| 云端分析失败 | 不覆盖原校准结果，提供重试和返回 |
| 分析超时 | 转为失败状态，提示检查网络后重试 |
| 文件拉取失败 | 保留原配置，提供重试或返回首页 |
| 文件格式无效/解析失败 | 不写入校准数据，提示文件不可用 |
| 重复提交 | 首次请求处理中忽略后续提交 |
| 用户按 Escape 取消 | 返回上一页或关闭弹窗，不提交当前未完成操作 |
| 删除取消 | 关闭弹窗，保留文件与生效状态 |
| 重置取消 | 关闭弹窗，保留全部当前设置 |
| 焦点丢失 | 页面恢复至默认控件或最近一次触发控件 |

## 8. 数据字段说明

| 字段 | 建议键名 | 类型 | 示例 | 规则 |
|---|---|---|---|---|
| 校准文件名 | `calibrationFileName` | String | `Camera_Calibration_20260720.cal` | 成功生成或导入后必填；管理页只读展示 |
| 导入时间 | `importedAt` | DateTime/String | `2026-07-20 14:30` | 使用电视本地时区；快速校准可记录结果生成时间 |
| 校准方式 | `calibrationMethod` | Enum | `QUICK` / `IMPORTED` | 与当前有效方式保持一致，只能取一个值 |
| 生效范围 | `effectiveScope` | Enum/Boolean | `ENABLED` / `DISABLED` | 原型以 Toggle 呈现；关闭不删除文件 |

### 8.1 辅助状态字段

| 字段 | 类型 | 说明 |
|---|---|---|
| `quickLevel` | Integer | 快速校准档位，取值 1–5；导入方式可为空 |
| `analysisStatus` | Enum | `IDLE`、`PROCESSING`、`SUCCESS`、`FAILED` |
| `importStatus` | Enum | `IDLE`、`LOADING`、`SUCCESS`、`FAILED` |
| `isApplied` | Boolean | 当前校准文件是否生效 |

## 9. 遥控器与焦点规则

- 方向键按控件空间位置移动焦点，不允许焦点移出当前页面可操作区域。
- 页面进入时必须提供默认焦点。
- `Enter` 激活当前控件，长按或重复触发不得产生重复任务。
- `Escape` 优先关闭 Dialog，其次返回上一页。
- Dialog 打开后启用焦点锁定，关闭后恢复至触发按钮。
